# Office 365 Icons for Windows
This repository contains a set of shortcuts to Microsoft Office Web 365 applications with custom icons.

The following icons are included:

- Word
- Excel
- Outlook
- PowerPoint

Each icon is linked to the respective application's homepage on Microsoft Office Cloud.

Instructions
To use these shortcuts, follow these steps:

1. Download and extract the ZIP file containing the shortcut files.
2. Copy the `365_icons` folder to your C:\ root.
3. Run `.\365_DesktopShortcut.ps1` from a powershell terminal withen the Users Profile you want the shortcuts.
4. (Optional) Right-click on each shortcut and select "Pin to taskbar" to add it to the taskbar.
